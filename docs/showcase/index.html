edu-role-play showcase

Live, in-browser role-plays generated with edu-role-play. Each is a single self-contained HTML file.

  • Manager performance feedback — Engineering manager giving difficult feedback to a junior team member.
  • Sales objection handling — Account exec navigating a price objection from a RevOps director.
  • Customer de-escalation — Support specialist handling an angry customer with a broken release.

Inference goes through a shared, rate-limited Cloudflare Worker. For your own deployments, run npx edu-role-play deploy-proxy.
